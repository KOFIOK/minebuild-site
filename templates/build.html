Способы установки

Выберите наиболее удобный для вас способ установки нашей сборки. Мы предлагаем несколько вариантов, от простого автоустановщика до архива для продвинутых пользователей.

Автоустановщик

Простой установщик, который автоматически настроит всю сборку за вас. Просто скачайте, запустите и следуйте инструкциям.

Удобный интерфейс Под любой лаунчер Импорт модов и настроек
~500 MB Временно недоступно
Рекомендуем

Modrinth Pack

Файл .mrpack для импорта в приложение или лаунчер Modrinth. Идеально для тех, кто уже использует Modrinth.

Простой импорт Автоматические обновления Лёгкая настройка
434 MB Скачать .mrpack

ZIP-архив

Полная сборка в виде ZIP-архива для ручной установки. Подходит для опытных пользователей или тех, кто хочет внести свои изменения.
